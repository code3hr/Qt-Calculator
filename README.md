# Qt-Calculator
Basic Qt GUI  calculator program.

## INSTALLATION
Make sure qt is installed
you can visit https://wiki.qt.io/Install_Qt_5_on_Ubuntu and follow the instructions

## usage
```shell
	$ git clone https://github.com/code3hr/Qt-Calculator.git
	$ cd the directory
	$ ./Calculator
```
